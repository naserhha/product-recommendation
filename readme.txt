A comprehensive product recommendation system for WooCommerce that helps customers find the best products based on their preferences and characteristics.

== Description ==

**Product Recommendation System** is a powerful WordPress plugin designed specifically for WooCommerce stores. It provides an intelligent product recommendation form that helps customers find the perfect products based on their preferences, needs, and characteristics.

== Features ==

* **Smart Product Recommendation Form**: Interactive questionnaire that guides customers to their ideal products
* **Fully Customizable Questions**: Add, edit, delete, and reorder questions from the admin panel
* **Multi-language Support**: Supports Persian/Farsi, English, Russian, Chinese, and Arabic
* **Modern Responsive Design**: Beautiful, mobile-friendly interface that works on all devices
* **Easy Integration**: Simple shortcode `[product_recommendation_form]` for placement anywhere
* **WooCommerce Integration**: Seamless integration with existing WooCommerce products
* **Customizable Appearance**: Adjust colors, fonts, and styling to match your brand
* **RTL Support**: Full support for right-to-left languages like Persian and Arabic
* **Random Product Fallback**: Shows random products when no exact matches are found
* **Admin Management**: Complete control over questions, products, and settings

== How It Works ==

1. **Configure Questions**: Set up custom questions in the admin panel
2. **Add Product Characteristics**: Assign characteristics to your WooCommerce products
3. **Place the Form**: Use the shortcode on any page or post
4. **Customer Experience**: Customers answer questions and get personalized recommendations
5. **Smart Matching**: The system matches customer preferences with product characteristics

== Localization ==

This plugin supports the following languages:
* Persian/Farsi (fa_IR, fa_AF, fa_TJ)
* English (en_US, en_GB, en_CA, en_AU, and more)
* Russian (ru_RU, ru_UA, ru_BY, ru_KZ, and more)
* Chinese (zh_CN, zh_TW, zh_HK, zh_SG, and more)
* Arabic (ar_SA, ar_EG, ar_AE, ar_JO, and more)

The plugin automatically detects your WordPress language setting and adapts all interface elements accordingly. You can translate the plugin into any other language using the included `.pot` files (`product-recommendation.pot` and `product-recommendation-new.pot`) in the `languages` directory.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/product-recommendation`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Product Recommendation System' in the admin menu
4. Configure your questions and settings
5. Add the shortcode `[product_recommendation_form]` to any page or post

== Frequently Asked Questions ==

= How do I add the recommendation form to my website? =

Use the shortcode `[product_recommendation_form]` on any page or post where you want the form to appear.

= Can I customize the questions? =

Yes! You can add, edit, delete, and reorder questions from the admin panel under 'Product Recommendation System > Settings > Base Questions'.

= Does this work with existing WooCommerce products? =

Yes, the plugin integrates seamlessly with your existing WooCommerce products. You just need to assign characteristics to your products.

= Is the form mobile-friendly? =

Yes, the form is fully responsive and works perfectly on all devices including mobile phones and tablets.

= Can I change the colors and styling? =

Yes, you can customize the primary color, font color, button color, and button text color from the settings panel.

= Does it support multiple languages? =

Yes, the plugin supports Persian/Farsi, English, Russian, Chinese, and Arabic with automatic language detection.
